A desktop-based financial management system developed using **Python**,**MySQL** and **CustomTkinter** to enable users to create their own accounts, login to them and add their financial goals. The app generates a financial plan suitable for the user. 

This project was developed as part of my **12th Standard Computer Science Project**.
This project also won the **Third place** out of **35** projects submitted in our School.
---

## Features

- Secure Login & Signup System (MySQL)
- Expense & Savings Tracking
- Goal-Based Financial Planning
- EMI Calculator
- Dynamic Financial Plan Generation
- Interactive GUI using CustomTkinter
- Integrated Financial Information Module (HTML)

---

## Technologies Used

- **Programming Language:** Python  
- **GUI Framework:** CustomTkinter  
- **Database:** MySQL  
- **Web Technologies:** HTML, CSS  
- **Libraries:** Pillow, Webbrowser, OS
